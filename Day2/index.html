HTML

HTML stands for Hypertext Markup Language and is a code used to structure and define the content of a webpage.The first version of HTML was written by Tim Berners-Lee. HTML tells a web browser how to display content, such as text, images and other media.It also labels content such as headings, paragraphs and links.

1)TAGS:

The Keyword which is enclosed between < and > to display the data in browser. Every tag has its own behaviour.
Types of Tags: 1) Paired tags 2) Unpaired tags

Paired tags

The paired tag which is having opening tag and closing tag with content.
ex: p, h1-h6, div, span, table etc...

UnPaired tags

The tag which closes itself is known as unpaired tag.
ex: img, br, hr, input etc...

2)Elements:

Combination of opening and closing tag with content between the tags. Elements can also have attributes.
Types of Elements: 1) Block Level element 2) Inline-block element 3) Inline Element

Block level element

The tag which occupies whole width and creates a new line.
ex: h1-h6, p, div, semantic elements etc...

Inline block element

The elements which occupies only required space and does not create new line and we can apply width, height, margin and padding properties.
ex: span, formatting, tags etc...

Inline element

The elements which occupies only required space and does not create a new line, in css we cannot apply width, height, padding top and bottom and margin top and bottom.
ex: img, iframe, input, button etc.....
